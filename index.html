Що таке GIT?

Git — це система контролю версій, яка дозволяє слідкувати за змінами в коді, зберігаючи “знімки” на різних етапах розробки.
Він допомагає розробникам співпрацювати, уникаючи конфліктів та зберігаючи історію змін.
Git не має графічного інтерфейсу за замовчуванням, але є численні інструменти, які допомагають візуалізувати роботу з ним.



Основні команди:

  1. git init — ініціалізує новий репозиторій Git в поточній теці.
  2. git add файл — додає зміни в певний файл до індексу.
  3. git commit -m “коментар” — фіксує зміни в репозиторії з коментарем.
  4. git push — відправляє зміни до віддаленого репозиторію.
  5. git pull — отримує зміни з віддаленого репозиторію.
  6. git clone — клонує віддалений репозиторій.
  7. git branch — управляє гілками в репозиторії.
